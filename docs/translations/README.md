## Contributing Translations

See [i18n Guide](../reference/i18n-guide.md) for detailed instructions on adding new languages.

### Quick Steps

1. Copy `docs/translations/template/en-US-template.ts` to `extensions/i18n/{locale}.ts`
2. Translate all strings
3. Register in `extensions/i18n/index.ts`
4. Run tests: `npm test`
5. Submit PR
